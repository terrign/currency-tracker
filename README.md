# Setup instructions:

1. Install [Node.js](https://nodejs.org/), if you haven't. Check if Node is installed:

```sh
node -v
```

2. Install packages:

```sh
yarn install
```

3. Run project locally:

```sh
yarn dev
```

# Scripts:

1. Build project:

```sh
yarn build
```

2. Start project locally:

```sh
yarn dev
```

3. Eslint ( <code>--fix</code> option is enabled):

```sh
yarn lint
```

4. Prettier ( <code>--write</code> option is enabled):

```sh
yarn prettier
```

5. Unit tests:

```sh
yarn test:unit
```

6. E2e tests:

```sh
yarn test:e2e
```

7. E2e open:

```sh
yarn cy:open
```
